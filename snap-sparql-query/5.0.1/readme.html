Snap SPARQL Query

A plug-in for querying ontologies with SPARQL-DL. This plug-in was developed for tutorial purposes as part of the Protege Short Course series. We make no quality guarantees regarding use in production environments.

Version info

v1.0.0 (17th Mar 2014)

  • Initial release
  • Autoupdate activated

v2.0.0 (26th Mar 2015)

  • Implementation of some builtins

v3.0.0 (28th Sep 2015)

  • Implemented more builtins
  • Queries involving Optional are now supported
  • Aggregate queries are now supported
  • Construct queries are now supported
v4.0.0 (20th Jan 2016)
  • Upgraded to OWL API version 4.1.3
v4.1.0 (18th Mar 2016)
  • Fixed rendering and highlighting problems that could occur on Windows
  • Added a sample query on startup
v4.2.0 (30th Nov 2016)
  • Fixed a problem that could result in an empty error message being displayed if there was no reasoner selected.
  • Fixed a problem with using untyped variables in a CONSTRUCT graph pattern
  • Added a custom builtin call named SPLITCAMEL that will split a camel case string into a string with separate words
v5.0.0 (31st Oct 2017)
  • Optimsed graph pattern matching especially for annotations
  • Implemented MD5, built in call
  • Several bug fixes
  • Fixed problems with DateTime built in calls
  • Arguments to built in calls are no longer implicitly cast
  • Added casting function support
v5.0.1 (13th Nov 2017)
  • Fixed a bug with comparing RDF terms (IRIs, Blank Nodes and non-string or non-numeric literals) using '='
